Doode

For When The Waves Are Swell

So How Does This All Work

Built during StirHack 2016, Doode is a service to notify you when waves at nearby beaches are poping.
It works by scraping webcam feeds like this. Using OpenCV we run some masking and clustering to find out the crest of the wave.

Using this data we run some calculations to work out what the waves are like and if they are surfable.
If the waves are good enough, we send everybody who's subscribed to the beach an SMS telling them that suf's up.
